# Auth Platform

A modern and scalable authentication platform built with ASP.NET Core and Angular following Clean Architecture principles.

Features:

✔ Email and password authentication  
✔ JWT and Refresh Tokens  
✔ Google Login  
✔ Facebook Login  
✔ Apple Sign-In  
✔ Role-based authorization (RBAC)  
✔ Dockerized environment  
✔ PostgreSQL integration  
✔ GitHub Actions CI/CD  
✔ Swagger API documentation  
✔ Enterprise-ready architecture

Tech Stack:

- ASP.NET Core 8
- Angular
- Entity Framework Core
- PostgreSQL
- Docker
- OAuth2 / OpenID Connect
- JWT Authentication
